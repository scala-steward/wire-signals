Companion object DispatchQueue

trait DispatchQueue extends ExecutionContext

A thin wrapper over Scala's scala.concurrent.ExecutionContext allowing us to differentiate between the default execution context which tries to run asynchronously as many tasks as possible, and limited execution contexts, allowed to run only up to a given number of tasks at once.
