    Creates a new button signal from the parent signal, a secondary signal indicating the state of a pressed/released button, and a function which performs a side-effect action every time the button signal is "clicked" on.

    V

    The parent signal.

    A secondary signal simulating a toggle or a button that can be pressed and released.
